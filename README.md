## 💡 About InsightIQ

InsightIQ is a professional AI-powered CSV Data Analyst platform built for data analysts, students, and business professionals.

**The Problem:** Most data analysts waste hours switching between Excel, SQL, and Power BI just to analyze a simple CSV file.

**The Solution:** InsightIQ — upload any CSV and get everything instantly. No SQL. No Power BI. No coding required.

---

## 🛠️ How It Was Built

### Phase 1 — Core Design & Development
The foundation of InsightIQ was designed and built from scratch using:
- **HTML, CSS, JavaScript** — complete frontend
- **Chart.js** — data visualization
- **Custom AI engine** — rule-based smart analysis (trends, anomalies, correlations, business insights)
- **5-Step guided analysis flow** — Upload → Clean → Analyze → Visualize → Export
- **AI Chat Assistant** — floating chat bubble for natural language queries
- **Dark professional UI** — Power BI inspired design

### Phase 2 — Enhancement with Enter Pro & Parcle
After the core was built, the project was further enhanced using:
- **Enter Pro (Converge AI)** — used as the AI-native development and deployment platform to scale and deploy the application
- **Parcle** — integrated as the persistent memory layer, allowing InsightIQ to remember every analysis session across visits

> 💬 *"Enter Pro helped deploy and enhance InsightIQ at production scale, while Parcle gave it memory — making it a truly sentient data analyst!"*

## 🎯 5-Step Analysis Flow

```
Step 1: Upload CSV
      ↓
Step 2: Clean Data (remove duplicates, fill missing values)
      ↓
Step 3: Analyze (statistics, trends, anomalies)
      ↓
Step 4: Visualize (auto charts — bar, line, pie, doughnut)
      ↓
Step 5: Export (cleaned CSV + full report)
```

Each step **automatically performs the action** and **explains what happened** — so even a non-technical user understands!

## 🧠 Parcle Memory Integration

InsightIQ uses **Parcle** as its persistent memory layer:
- Every question asked is saved to Parcle cloud
- Every insight generated is stored with metadata
- On next session, full history is restored automatically
- Agent gets smarter with every use
